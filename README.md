# Murder-at-Blackthorn-Manor
Murder at the Blackthorn Manor is an interactive detective mystery game built with React.js. Players investigate the murder of Edward Blackthorn by exploring a crime scene, collecting evidence, interrogating suspects, and analyzing clues to identify the killer. 

The game features interactive hotspots, progressive clue unlocking, dynamic interrogations based on collected evidence, a corkboard-style Evidence Board, notebook-style Case Files, immersive background music, and a final accusation system. 

Developed using React.js, JavaScript, HTML, CSS, React Router, Context API, and Vite, this project demonstrates component-based architecture, state management, routing, and interactive UI design while delivering an engaging mystery-solving experience.
